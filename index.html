Parcijalni ispit BackendDeveloper 1/5

Zadatak ispita

  • kreirati virtualni stroj
  • instalirati Ubuntu Linux OS
  • instalirati Apache server, MySQL, PHP i Composer
  • postaviti /var/www/html/ParcijalniIspitmapu kao root mapu servera
  • inicijalizirati GIT
  • kreirati i povezati udaljeni repozitorij (ParcijalniIspitBD1)
  • instalirati bilo koji paket s Packagist
  • kreirati .gitignore datoteku i dodati mapu /vendor
  • napraviti inicijalni commit i promjene spremiti na udaljeni repozitorij
